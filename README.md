## Como iniciar un proyecto de Python

1. Crea un entorno virtual con `python3 -m venv venv`.
2. Activa el entorno con `source venv/bin/activate`.
3. Ejecuta `python hola_mundo.py` para ver el mensaje en la consola.
4. Ejecuta `python -m unittest` para correr las pruebas.
